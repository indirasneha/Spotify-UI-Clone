# Spotify UI Clone

A recreation of the Spotify Web Player user interface built using **HTML and CSS**. This project focuses on replicating the layout and design of Spotify’s web interface while practicing **Flexbox layouts, UI structuring, and responsive styling**.

## Live Demo

https://indirasneha.github.io/Spotify-UI-Clone/

## GitHub Repository

https://github.com/indirasneha/Spotify-UI-Clone

## Features

* Spotify-style sidebar navigation
* Sticky top navigation bar
* Playlist and music recommendation cards
* Music player interface with playback controls
* Styled progress bar and volume slider
* Layout built using Flexbox

## Tech Stack

* HTML5
* CSS3
* Font Awesome
